## 技术类分享

### 轻量级JavaScript运行时

[https://antjs.org](https://antjs.org)

Ant是一个全新的轻量级JavaScript运行时，整个二进制文件仅9MB。它从零构建在自研引擎之上，支持运行真实npm包和TypeScript，提供Node兼容API、VM隔离沙箱和内置Wasm支持。作为Deno和Bun之外的又一JavaScript运行时选择，Ant主打极致轻量和安全隔离，适合边缘计算和嵌入式场景。

### git history

[https://lalitm.com/post/git-history/](https://lalitm.com/post/git-history/)

git 2.54和2.55版本新增的实验性命令git history，认为它值得更多关注。该命令提供了reword、split和fixup子命令，能在不切换工作流的前提下解决并行开发中频繁rebase带来的痛点。作者认为git history已能提供jj（git替代工具）的部分核心优势，且无需额外安装，是核心git发行版的一部分。

### 涟漪鱼塘

[https://seanwong17.github.io/RippleAquarium/](https://seanwong17.github.io/RippleAquarium/)

纯前端 WebGL 水族箱，鱼群会聚集、转向、互相避让，并有实时水面涟漪，[代码开源](https://github.com/SeanWong17/RippleAquarium)。

### C语言小书

[https://little-book-of.github.io/c/books/en-US/book.html](https://little-book-of.github.io/c/books/en-US/book.html)

一本面向初学者的 C 语言小教程。适合刚进入大学的大学生。

## 非技术类分享

### 怎样读更多的书

[https://scotto.me/blog/2026-07-12-how-to-read-more-books/](https://scotto.me/blog/2026-07-12-how-to-read-more-books/)

作者分享了自己从一年读不到十本书到每周一本的阅读方法。核心策略包括：用书替代手机填充碎片时间（删除社交和流媒体App）、随身携带电子书阅读器、同时阅读多本书以保持新鲜感、广泛涉猎不同类型、不害怕弃读（有些书需要对的时机再读），以及建立个人"反图书馆"——未读的书比已读的更能代表知识边界。

### 沉浸式日语学习网页应用

[https://jivx.com/densha](https://jivx.com/densha)

这是一个沉浸式日语学习网页应用，用户可以在与日本真实时间、天气和季节同步的体素风格东京中"乘坐"山手线。应用播放lo-fi背景音乐，同时朗读JLPT N5级别的日语句子并以字幕形式飘过画面，打造出一个氛围感十足的日语环境学习空间，用户只需按下播放即可沉浸其中。

### 我们是否将过多的思考工作交给人工智能

[https://www.artfish.ai/p/offloading-thinking-to-ai](https://www.artfish.ai/p/offloading-thinking-to-ai)

第一次听说连自己想吃什么都询问AI，这样的思考太依赖与AI了，使人类进入到非思考模式，这样的人还是自己吗？所谓为人就是因为每个人都有差异性，且独一无二，如何使用AI是我们的选择，当你过分依赖一件事物的时候，那就你就不再是你，你没有思考和解决问题的能力。
